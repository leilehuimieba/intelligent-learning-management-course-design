# 智能学习管理程序答辩说明

## 1. 项目定位

- 项目名称：智能学习管理程序课程审核版
- 技术路线：Java + Activity + XML + SQLite
- 适用场景：面向学生的任务管理、专注计时与学习统计

本版本专门面向《Android开发技术》课程审核要求设计，强调：

1. 使用 Java 语言进行原生 Android 开发。
2. 以 Activity 页面跳转为主。
3. 使用本地 SQLite 数据库存储任务与专注记录。
4. 提供可现场演示的完整 APK。

## 2. 已实现功能

### 2.1 首次引导

首次打开应用进入引导页，填写：

1. 昵称
2. 每日目标分钟数
3. 默认专注时长

保存后写入 `SharedPreferences`，再次启动会直接进入首页。

### 2.2 任务管理

任务管理页支持：

1. 新建任务
2. 查看任务列表
3. 标记任务完成

数据保存在 SQLite 的 `study_task` 表中。

### 2.3 专注计时

专注页支持：

1. 输入专注时长
2. 开始倒计时
3. 手动结束并保存记录
4. 自动完成后保存记录

数据保存在 SQLite 的 `focus_session` 表中。

### 2.4 统计分析

统计页可查看：

1. 累计专注时长
2. 已完成任务数量
3. 专注会话次数

### 2.5 设置与导出

设置页支持修改昵称、每日目标和默认专注时长。  
首页支持将当前汇总数据导出为 CSV 文件，并可通过系统分享面板发送。

## 3. 数据库设计

### 3.1 study_task 表

字段：

1. `id`
2. `title`
3. `category`
4. `estimated_minutes`
5. `status`
6. `created_at`

### 3.2 focus_session 表

字段：

1. `id`
2. `planned_minutes`
3. `actual_minutes`
4. `started_at`
5. `ended_at`

## 4. 页面流程

1. 启动应用
2. 首次进入引导页
3. 保存基本信息后进入首页
4. 从首页进入任务管理、专注计时、统计分析、设置页面
5. 在首页导出 CSV 周报

## 5. 现场演示建议

答辩时建议按下面顺序演示：

1. 打开应用，说明使用 Java + Activity + XML 开发
2. 展示首次引导并保存参数
3. 进入任务页，新增一个任务并标记完成
4. 进入专注页，开始一个短时专注并结束
5. 进入统计页查看数据变化
6. 回到首页导出 CSV 并展示文件名

## 6. 可以主动说明的亮点

1. 页面结构清晰，便于老师检查 Activity 跳转流程。
2. SQLite 本地存储满足课程对数据库增删改查的要求。
3. 输入有合法性校验，非法输入会提示用户。
4. 支持 CSV 导出，增强了完整性和可展示性。

## 7. 当前版本边界

当前课程审核版优先满足教学要求，因此重点放在：

1. 原生 Java 页面结构
2. 本地数据库
3. 可演示的完整功能链路

未引入复杂网络、依赖注入和 Compose 等高级框架，以降低审核争议。
