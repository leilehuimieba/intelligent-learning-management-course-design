# 项目截图目录

该目录用于存放仓库首页 README 展示用的项目截图。

## 建议截图内容

- `home.png`：首页
- `onboarding.png`：首次使用引导页
- `task.png`：任务管理页
- `focus.png`：专注计时页
- `stats.png`：统计分析页
- `settings.png`：参数设置页

## 使用方式

当你补充截图后，可在仓库根目录 `README.md` 中使用如下方式引用：

```md
![首页截图](docs/assets/screenshots/home.png)
```

## 说明

当前该目录仅作为截图占位与后续补充入口使用。
